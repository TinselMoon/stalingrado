## Como compilar

### Dependências
- CMake 3.x+
- G++ ou clang++
- SFLM 2.6.2

### Build
```bash
mkdir build
cd build
cmake ..
make
```

-Operating System:

Linux 6.17 and higher (not guaranteed to run properly on older versions).

-Compiler:
